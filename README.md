# Wine Quality Analysis: Classification and Regression

This project analyses red and white wine quality using physicochemical properties. Both classification and regression approaches are applied.

## Objective

- Classification: predict wine quality categories: Low, Medium, High
- Regression: predict the original wine quality score

## Models Used

### Classification
- Gaussian Naive Bayes
- Logistic Regression
- Random Forest Classifier

### Regression
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Dataset

The dataset contains red and white wine samples with physicochemical features such as acidity, sugar, chlorides, sulfur dioxide, density, pH, sulphates, alcohol, and quality score.

Source: UCI Wine Quality Dataset.
