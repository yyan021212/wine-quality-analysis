# Wine Quality Analysis: Classification and Regression

This project analyses red and white wine quality using physicochemical properties. Both classification and regression approaches are applied to compare how different machine learning models predict wine quality.

## Project Structure

```text
wine-quality-analysis/
├── WineQuality_RED.ipynb
├── WineQuality_White.ipynb
├── wine+quality.zip
├── README.md
└── requirements.txt

The dataset ZIP file is already included in this repository, and the notebooks contain saved outputs,
so the results can be viewed directly on GitHub without manually extracting the ZIP file or rerunning the code.
